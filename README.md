First at all, install [NodeJs](https://nodejs.org/en) and [Git](https://git-scm.com/)

## Run

``` git clone https://github.com/AndersonAlencarBarros/How2uSleepRepo.git```

```cd How2uSleepRepo```

Now on the folder, run the following commands:

```npm install```

```npm run build```

```npm start```
